# 메시지 숨김 버튼 구현
+ 개발 기간 : 2023.07.15 ~ 2023.11.21(약 4개월)
+ 개발 인원 : 3명

## 배경
+ 여러 채팅앱을 봤는데 비밀채팅방을 따로 개설하여 숨겨주는 기능은 있었지만, 메시지별 숨김(잠금) 기능은 없기에 기획하게 됐습니다.

## 개발 환경 및 적용 기술
+ Android Studio
+ RealTime DataBase
+ 사용 언어 : JAVA

## 구현 기능
+ 회원 가입 기능
+ 로그인 기능
+ 채팅 기능
+ 메시지별 잠금 기능
